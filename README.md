# Projeto-BeeTech-Salve-as-Abelhas-
Projeto Integrador 2
Este projeto integra design arquitetônico, sustentabilidade e preservação da biodiversidade através do desenvolvimento de caixas-maquete personalizadas para abelhas nativas sem ferrão. 
O foco principal é transformar a meliponicultura urbana em um objeto de design exclusivo, promovendo a regeneração ambiental por meio da venda de kits de sementes e mudas de plantas melitófilas.

# Visão Geral do Projeto
Diferente das caixas padrão de mercado, o Bee-Hub oferece réplicas arquitetônicas detalhadas (maquetes em escala) que servem como "casca" protetora para ninhos de abelhas nativas. O modelo de negócio abrange:

Design Customizado: Réplicas de prédios, residências ou instituições (como o campus IFPR) feitas em madeira balsa e acrílico.

Bem-estar Animal: Sistema de "caixa dupla" que garante isolamento térmico e proteção contra luminosidade.

Kit de Regeneração: Venda conjunta de sementes e mudas de flores específicas para atrair polinizadores locais.

# Materiais e Tecnologias
Para o desenvolvimento deste projeto, são utilizados:

Softwares de Design: AutoCAD, Civil 3D e SketchUp para o detalhamento das fachadas e estruturas.

Materiais Físicos: Madeira balsa para a maquete externa, madeiras sustentáveis para o ninho interno e chapas de acrílico para janelas de observação não invasiva.

Logística Verde: Embalagens biodegradáveis para o envio de mudas e sementes.
